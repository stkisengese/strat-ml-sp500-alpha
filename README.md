# strat-ml-sp500-alpha
Strat-ML: S&amp;P 500 Alpha Generation Framework:- This repository contains a complete quantitative pipeline designed to outperform the S&amp;P 500 Index using Machine Learning. The project focuses on out-of-sample signal generation using constituent-level OHLCV data, rigorous Blocking Time Series Cross-Validation, and a Long/Short Stock Picking strategy.
